# .vim

My .vim folder since 2009.

## Install

    cd
    git clone https://github.com/imwilsonxu/.vim
    ln -s ~/.vim/.vimrc ~/.vimrc
    git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
    # Open Vim and Install plugins.
    vim
    :BundleIntall
